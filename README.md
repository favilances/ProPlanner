### What is ProPlanner?

ProPlanner is a **fully client-side calendar application** built with Vue 3. All your data stays in your browser - no servers, no databases, complete privacy.

### How It Works

1. **Create Events**: Click on any date or use the "Add Event" button
2. **Edit & Manage**: Drag, drop, resize events directly on the calendar
3. **Multi-Language**: Switch between English, Turkish, and Spanish - calendar automatically adapts
4. **Theme Support**: Toggle between Light and Dark themes
5. **Export**: Download your calendar as `.ics` file for use in any calendar app
6. **Privacy First**: Everything is stored in your browser's localStorage - no external servers

### Tech Stack

- **Vue 3** - Modern reactive framework
- **FullCalendar** - Professional calendar component with locale support
- **TailwindCSS** - Utility-first styling
- **Vite** - Fast build tool
- **LocalStorage** - Client-side data persistence

### Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.
